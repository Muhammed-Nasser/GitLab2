GitLab2

1.To Remove a local branch
-
   git branch -d branch_name
-
2.To delete local tags
-
   git tag -d v1.0
-
3.To list all tags
-
   git tag 
-
4.To delete local tags
-
   git tag -d tag_name
-
5.To delete remote tag
-
   git push origin --delete tag_name
